# 📊 Dashboard de Asistencia Semanal – Power BI

Este proyecto es un dashboard interactivo desarrollado en Power BI para visualizar y analizar la asistencia semanal del personal en una empresa.

---

## 💼 Business Understanding

El área de Recursos Humanos necesitaba una herramienta visual y de fácil lectura para monitorear diariamente la asistencia del personal.  
El objetivo principal era identificar patrones de ausentismo y tardanzas por departamento y facilitar la toma de decisiones mediante un resumen claro y accesible para supervisores y directivos.

---

## 📈 Data Understanding

- **Fuente de datos:** Archivo de Excel con registros de entradas diarias por colaborador.
- **Campos clave:** Nombre del colaborador, fecha y hora de entrada, departamento.
- **Volumen de datos:** Información semanal simulada (puede escalarse a datos históricos).

Se realizó limpieza de datos para unificar formatos de fecha y hora, y se creó una columna categórica para clasificar si una entrada era puntual, tardía o ausencia.

> 🛡️ *Nota: Todos los datos presentados en este proyecto son ficticios y fueron generados exclusivamente con fines ilustrativos, en resguardo de la confidencialidad del cliente.*

---

## 🧠 Approach

1. **Preparación de datos:**  
   - Limpieza y transformación de columnas (fechas, horarios, categorías).
   - Cálculo de KPIs clave como porcentaje de puntualidad y total de ausencias.

2. **Modelado en Power BI:**  
   - Relación directa con una tabla calendario para análisis temporal.
   - Uso de medidas DAX para calcular métricas dinámicas.

3. **Visualización:**  
   - Diseño orientado a claridad y acción rápida.
   - Filtros de fecha y nombre para análisis personalizado.
   - Visuales principales: tarjetas KPI, barras, líneas y segmentadores.

---

## 🛠️ Tecnologías utilizadas

- **Power BI Desktop** – Plataforma principal para modelado y visualización.  
- **Microsoft Excel** – Fuente de datos utilizada para la simulación.  
- **Lenguaje DAX (Data Analysis Expressions)** – Para creación de medidas personalizadas y KPIs dinámicos.  
- **Power Query** – Limpieza y transformación de datos desde el editor integrado.  
- **Tabla de fechas personalizada** – Para análisis temporal detallado.

---

## 🧩 ¿Qué muestra el dashboard?

- ✅ Total de colaboradores
- 📉 Asistencias, ausencias y tardanzas por departamento
- 📆 Tendencia diaria de asistencias
- 🔍 Filtros por nombre y fecha para análisis individual
- 📌 Porcentaje de ausencias

---

## 🚀 ¿Por qué lo desarrollé?

Inicialmente fue un bosquejo para mostrar un ejemplo rápido al equipo de Recursos Humanos, pero terminó siendo una herramienta muy bien recibida por su utilidad y claridad.  
Este dashboard no se queda aquí: será automatizado para que el equipo tenga acceso a datos actualizados en tiempo real y pueda tomar decisiones informadas con mayor rapidez.

# 📊 Weekly Attendance Dashboard – Power BI

This project is an interactive dashboard developed in Power BI to visualize and analyze weekly employee attendance in a corporate environment.

---

## 💼 Business Understanding

The Human Resources team needed a clear and easy-to-read tool to monitor employee attendance on a daily basis.  
The main goal was to identify patterns of absenteeism and tardiness by department and facilitate decision-making through a concise, accessible summary for supervisors and managers.

---

## 📈 Data Understanding

- **Data source:** Excel file with daily employee check-in records.  
- **Key fields:** Employee name, check-in date and time, department.  
- **Data volume:** Simulated weekly data (scalable to historical datasets).

The dataset was cleaned to unify date and time formats, and a categorical column was created to classify entries as on-time, late, or absent.

> 🛡️ *Note: All data used in this project is fictitious and was generated exclusively for illustrative purposes to ensure client confidentiality.*

---

## 🧠 Approach

1. **Data Preparation:**  
   - Cleaning and transforming date, time, and category columns.  
   - Calculating KPIs such as punctuality rate and total absences.

2. **Power BI Modeling:**  
   - Custom date table for time-based analysis.  
   - DAX measures to create dynamic metrics.

3. **Visualization:**  
   - Clear, decision-focused layout.  
   - Slicers for name and date to enable individual-level filtering.  
   - Visuals used: KPI cards, bar charts, line graphs, and slicers.

---

## 🛠️ Technologies Used

- **Power BI Desktop** – Main platform for modeling and visualization.  
- **Microsoft Excel** – Simulated data source.  
- **DAX (Data Analysis Expressions)** – For custom calculations and dynamic KPIs.  
- **Power Query** – For data transformation and cleaning.  
- **Custom date table** – For temporal analysis.

---

## 🧩 Dashboard Features

- ✅ Total employee count  
- 📉 Attendance, absence, and tardiness by department  
- 📆 Daily attendance trend  
- 🔍 Filters by employee name and date  
- 📌 Absenteeism percentage

---

## 🚀 Why I built it

This started as a quick mock-up to support the HR team, but it ended up being a very well-received tool thanks to its simplicity and value.  
This dashboard will be automated so that the team can access updated data in real-time, streamlining their daily tasks and improving their decision-making.
